# 釐清問題

PracticeCard.tips 欄位是否固定為三項？

# 定位

ERM: PracticeCard.tips[3]

# 多選題

| 選項 | 描述 |
|---|---|
| A | 是，固定為 3 項 |
| B | 否，是可變動數量的列表，最多 3 項 |
| C | 否，是可變動數量的列表，無上限 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響練習卡（PracticeCard）的資料結構定義與前端顯示邏輯。如果為可變數量，需考慮 UI 如何適應不同數量的提示。
