# 釐清問題

Symptom 的 synonyms 列表最大長度限制為何？

# 定位

ERM：Symptom 表的 synonyms 屬性，該屬性為 text[] 型別

# 多選題

| 選項 | 描述 |
|--------|-------------|
| A | 最多 10 個同義詞 |
| B | 最多 20 個同義詞 |
| C | 最多 50 個同義詞 |
| D | 不設上限，由實際需求決定 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響資料庫 schema 設計，資料驗證邏輯，以及後端 API 處理同義詞的邏輯
