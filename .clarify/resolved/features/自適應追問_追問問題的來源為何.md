# 釐清問題

用於追問的 0 到 2 個問題從何而來？

# 定位

Feature: 3. 功能需求（FR，最小集合）- 2) 自適應追問

# 多選題

| 選項 | 描述 |
|---|---|
| A | 每個「症狀種子」預先綁定一組追問問題 |
| B | 一個全域的、通用的追問問題庫 |
| C | 根據缺失的 Slot 資訊（等級/地形）動態生成問題 |
| D | 由大型語言模型根據上下文即時生成 |
| Short | 提供其他簡短答案（<=5 字）|

# 影響範圍

影響追問的相關性與品質。預綁定的問題（A）較可控，但缺乏彈性；即時生成（D）彈性高，但可能產生不穩定的結果。

# 優先級

High
